Objectives

Our platform aims to automate the manual processes of:

  • Advertising room/hostel/apartment rentals
  • Buying and selling used items among university students

Project Type

Non-profit, Social Service, Automation

Key Features

  • The registration system requires student ID for Khulna University students and addresses for outsiders.
  • Email-based login system.
  • Information on room/hostel/apartment includes:
    • Number of rooms available
    • Number of people needed
    • Floor
    • Location
    • Photo of building and room
    • Photo of washroom
    • Total rent cost (excluding bills and charges)
    • Details on electricity, gas, water, security, and internet bills
    • Room size
    • Garage availability
    • Balcony size
  • Advertisements are removed after transaction approval.
  • Ads that are 15 days old automatically become inactive and won't be shown in the feed, but can be accessed via ad ID or from the user dashboard.
  • Dashboard for the maintainer/admin to inspect:
    • Warned users
    • Reported users
    • Reported ads
    • Pending ads waiting for approval
  • Banned users have their profile highlighted in red.
  • Moderators can see warning and ban buttons on user profiles (except their own).
  • Banned users are shown a notice when attempting to log in.
  • Restriction to prevent admins/moderators from warning or blocking themselves.
  • The admin can add and remove moderators.
  • Users can create ads for room/mess rentals and selling used items.
  • Ad renewal feature: Users can renew ads and see renewal status in their profile's ad list.
  • Favorites/marking system for ads.
  • Any user can comment on any post.
  • Reporting system: Reported ads are highlighted (font color set to red).
  • Announcements are fetched from a text file and displayed.
  • Ad details include:
    • Room count
    • Student count
    • Available from
    • Agreement policy
    • Bathroom details
    • Roommate details
    • Location link (map)
    • Religion preference
    • Security
    • Furniture
    • Entry time
    • Owner‚Äôs name/contact
    • Distance from KU
    • Facing side
    • Kitchen/fridge/drinking water/balcony/room size
    • Nearby landmarks
    • Utility bills and facilities detailed per ad (food, CCTV, geyser, IPS, Wi-Fi, electricity, garbage, fridge, security, assistant, etc.)
  • Prevention of invalid user account access/manipulation via direct URL.
  • Feedback/error handling in form submissions.
  • Highlighting/restricting access as per user roles and statuses (banned, moderator, admin).
  • Clear user feedback via UI highlights (red color for banned/reported).

Current Status

The project is currently under development. We are working on creating a platform that will significantly improve the housing search and used item marketplace experience for KU students.
