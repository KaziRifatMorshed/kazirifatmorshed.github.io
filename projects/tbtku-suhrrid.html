tbtKU Suhrrid (সুহৃদ)

Suhrrid (সুহৃদ) is a Student Service Platform for the students of Khulna University, designed to be a well-wisher and benefactor for students seeking accommodations and purchasing used items.

Objectives

Our platform aims to automate the manual processes of:

  • Advertising room/hostel/apartment rentals
  • Buying and selling used items among university students

Project Type

Non-profit, Social Service, Automation

Key Features

  • The registration system requires student ID for Khulna University students and addresses for outsiders.
  • Email-based login system.
  • Information on room/hostel/apartment includes detailed attributes like availability, floor, rent, bills, size, garage, etc.
  • Advertisements are removed after transaction approval.
  • Automatic ad expiration and renewal feature.
  • Dashboard for maintainer/admin to inspect users and ads.
  • Reporting and favorites system.
  • Interactive commenting system.

Current Status

The project is currently under development. We are working on creating a platform that will significantly improve the housing search and used item marketplace experience for KU students.
